    9. 重绘和回流
        重绘 不会影响页面布局操作，比如修改颜色
        回流 布局的改变导致需要重新构建，就是回流
        回流必将引起重绘，重绘不一定引起回流

    11. 为什么要初始化css样式
          应为浏览器的兼容问题，不同浏览器对标签的默认值是不同的，如果不初始化css,会导致不同浏览器页面中的显示差异
    
    12. 居中为什么用 transform ? 为什么不用 margin-left/margin-top
          transform 属于合成属性，不会引起整个页面的回流重绘，节省性能消耗，但是占用内存会大一些
          margin-left/margin-top 属于布局属性，会引起页面的 layout回流和repaint重绘

    13. 什么是BFC 块级格式化上下文
          块级元素参与的布局，BFC布局将自己与外部隔开，内部外部互不影响
          
          触发BFC
            body 根元素
            浮动元素 flaot 不为 none以外的值
            绝对定位元素
            display 为 inline-block,table-cells,flex
            overflow 除了visible 以外的值 （hidden,auto,scoll）

          BFC的作用
            同一个BFC下边距会发生重叠
              如果想要避免外边距发生重叠，可以放在不同的BFC容器中
            BFC可以包含浮动的元素
              浮动的元素会脱离文档流，父级使用overflow:hidden;触发BFC可以撑起父级使其包含浮动元素
            BFC可以阻止元素被浮动元素覆盖
              同级的两个元素其中一个浮动会覆盖没有浮动的元素，要想不被覆盖，没有浮动的元素可以设置 overflow:hidden

    15. 标准盒子模型，IE盒子模型
          标准盒子大小 = content + border + margin + padding  
            盒子的宽等于content的宽
            盒子的高等于content的高
          IE盒子大小 = (content + border + padding)  + margin
            盒子的宽等于 content的宽 + 左右border + 左右padding
            盒子的高等于 content的高 + 上下border + 左右padding
